
### Configuração do Ambiente

1. **Baixe e instale:**
   - [Node.js (versão LTS recomendada)](https://nodejs.org/)
   - [Python 3.11 ou superior](https://www.python.org/downloads/)

2. **Verifique se os binários estão funcionando:**
   ```bash
   node --version
   npm --version
   python --version
   ```

### Clone o Repositório

```bash
git clone https://github.com/seu-repositorio.git
cd seu-repositorio
```

### Configuração do Backend

#### Configurar o Backend para a IA (serverIA):
1. **Entre na pasta do backend para IA:**
   ```bash
   cd AI
   ```

2. **Crie um ambiente virtual Python:**
   ```bash
   python -m venv .venv
   ```

3. **Ative o ambiente virtual:**
   - **Windows (PowerShell):**
     ```bash
     .venv\Scripts\Activate
     ```
   - **Mac/Linux:**
     ```bash
     source .venv/bin/activate
     ```

4. **Instale as dependências do Python:**
   ```bash
   pip install -r requirements.txt
   ```

5. **Certifique-se de instalar FFmpeg para conversão de vídeos:**
   - **Windows:** Baixe o FFmpeg em [ffmpeg.org](https://ffmpeg.org/), configure no `PATH`.
   - **Linux/Mac:** 
     ```bash
     sudo apt update && sudo apt install ffmpeg
     ```

6. **Inicie o servidor da IA:**
   ```bash
   node serverIA.mjs
   ```

#### Configuração do Backend de Usuários (server.js):

1. **Entre na pasta do backend de usuários:**
   ```bash
   cd backend
   ```

2. **Instale as dependências do Node.js:**
   ```bash
   npm install
   ```

3. **Inicie o servidor:**
   ```bash
   node server.js
   ```

### Configuração do Frontend

1. **Entre na pasta do frontend:**
   ```bash
   cd frontend
   ```

2. **Instale as dependências do Node.js:**
   ```bash
   npm install
   ```

3. **Inicie o servidor do frontend:**
   ```bash
   npm run dev
   ```

### Teste a Aplicação
1. Certifique-se de que o **server.js** (usuários), **serverIA.mjs** (IA) e o **frontend** estejam rodando ao mesmo tempo.
2. Acesse o frontend no navegador em `http://localhost:3000`.

### Comandos Rápidos de Reinstalação

```bash
# Backend IA
cd AI
python -m venv .venv
.venv\Scripts\Activate # (ou source .venv/bin/activate no Linux/Mac)
pip install -r requirements.txt

# Backend Usuários
cd backend
npm install

# Frontend
cd frontend
npm install
```

### Problemas Comuns
- **Erro ao rodar o Python:** Certifique-se de que o Python está configurado no `PATH`.
- **Dependências ausentes:** Reinstale com `pip install` ou `npm install` conforme necessário.
- **Erro de vídeo:** Confirme que o FFmpeg está configurado corretamente no sistema.
